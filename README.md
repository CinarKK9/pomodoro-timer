# pomodoro-timer

# How to use

1. Clone the repository
2. run the following command in the terminal

```bash
$ cd pomodoro-timer
$ npx tailwindcss -i ./src/input.css -o ./src/output.css
```

3. open the html file in your browser
4. Start the timer
